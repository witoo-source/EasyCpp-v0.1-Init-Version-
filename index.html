Características de EasyCpp

Interfaz sencilla

Utiliza una interfaz simple y clara que facilita su uso para todos los niveles de experiencia.

Funciones útiles

Contiene funciones útiles como suma, resta, multiplicación, división y más.

Compatibilidad

Compatible con todas las versiones modernas de C++ y es fácilmente integrable en proyectos existentes.

Uso de EasyCpp

EasyCpp es muy fácil de integrar en tu proyecto. Aquí tienes un ejemplo básico:

#include "init.h"

int main() {
    init calc(1);

    // Ejemplo de uso
    int resultado = calc.sum(5, 3);
    calc.write("El resultado es: ").write(resultado).endline();

    return 0;
}
